# RIASEC 가중치 매핑 적절성 분석 보고서

## 1. 분석 개요

### 1.1 분석 대상
- **현재 매핑**: `majorList.ts` (281-454 라인)의 전공별 RIASEC 가중치
- **참고 데이터**:
  - `job_code_inform.csv`: 실제 직무의 RIASEC 데이터
  - `majorcapacity.csv`: 전공별 역량 정보
- **목표**: 50개 전공을 효과적으로 구분할 수 있는지 평가

### 1.2 RIASEC 차원
- **R** (Realistic): 현실형
- **I** (Investigative): 탐구형
- **A** (Artistic): 예술형
- **S** (Social): 사회형
- **E** (Enterprising): 진취형
- **C** (Conventional): 관습형
- **V** (Values): 가치 (명지대 특화 차원)

## 2. 현재 매핑의 주요 특징

### 2.1 계열별 RIASEC 패턴

#### 인문대학 (Humanities)
```
주요 패턴:
- 언어/문학 전공: A (0.8-0.85), I (0.7-0.75), S (0.6-0.75)
- 사학/미술사: I (0.85-0.9), C (0.7-0.8), A (0.6-0.8)
- 문헌정보학: C (0.9), I (0.8)
- 철학: I (0.9), V (0.85), A (0.6)
- 문예창작: A (0.95), I (0.7)
```

**평가**: 인문대학 내 전공 간 차별화가 어느 정도 이루어지고 있음
- 문헌정보학의 높은 C (0.9) 값은 적절
- 사학/미술사의 I 중심 배치는 적절
- 문예창작의 A (0.95) 강조도 적절

#### 사회과학대학 (Social Sciences)
```
주요 패턴:
- 행정학: C (0.9), S (0.75), V (0.7)
- 정치외교학: E (0.85), V (0.8), S (0.75)
- 경제학: E (0.9), I (0.8), C (0.75)
- 국제통상학: E (0.9), S (0.75), I (0.7)
- 응용통계학: I (0.95), C (0.9)
- 법학: C (0.95), I (0.8), E (0.65)
```

**평가**: 사회과학대학 내 전공 간 차별화가 잘 이루어짐
- 법학의 C (0.95) 강조는 적절 (실무적 특성)
- 응용통계학의 I (0.95), C (0.9)는 수학적 특성 반영
- 정치외교학의 E (0.85) 강조는 적절

#### 경영대학 (Business)
```
주요 패턴:
- 경영학: E (0.95), I (0.75), C (0.8)
- 글로벌비즈니스학: E (0.95), S (0.8), I (0.7)
- 경영정보학: I (0.9), C (0.85), E (0.75)
```

**평가**: 경영대학 내 전공 간 차별화는 보통
- 경영학과 글로벌비즈니스학의 차별화 부족 (둘 다 E 0.95)
- 경영정보학의 I (0.9) 강조는 적절 (IT 특성)

#### 공과대학 (Engineering)
```
주요 패턴:
- 컴퓨터/정보통신: I (0.9-0.95), C (0.8-0.85), R (0.65-0.7)
- 전기/전자: I (0.9), R (0.85-0.9), C (0.8)
- 기계/로봇: R (0.95), I (0.85-0.9), C (0.8-0.85)
- 화학/신소재: I (0.9), R (0.85), C (0.8)
- 건설/환경: R (0.85-0.9), I (0.75-0.8), C (0.75-0.8)
```

**평가**: 공과대학 내 전공 간 차별화는 양호
- 기계공학의 R (0.95) 강조는 적절
- 컴퓨터공학의 I (0.95) 강조도 적절
- 하지만 일부 공학 전공들 간 차별화 부족

#### 예술/스포츠대학 (Arts & Sports)
```
주요 패턴:
- 디자인 전공: A (0.9-0.95), E (0.7-0.8), I (0.65-0.75)
- 음악 전공: A (0.95), S (0.8-0.85), E (0.75-0.8)
- 연극/뮤지컬: A (0.95), S (0.9), E (0.85)
- 체육학: R (0.95), S (0.9), E (0.8)
- 스포츠산업학: E (0.9), S (0.85), R (0.75)
```

**평가**: 예술/스포츠대학 내 전공 간 차별화는 양호
- 디자인의 A (0.9-0.95) 강조는 적절
- 체육학의 R (0.95) 강조도 적절
- 스포츠산업학의 E (0.9) 강조는 비즈니스 특성 반영

## 3. 문제점 및 개선 필요 영역

### 3.1 유사한 프로필을 가진 전공 그룹

#### 그룹 1: 언어/문학 전공 (차별화 부족)
```
중어중문학전공: A(0.8), I(0.75), S(0.65)
일어일문학전공: A(0.8), I(0.75), S(0.65)  ← 동일!
국어국문학전공: A(0.85), I(0.75), S(0.6)  ← 거의 동일
영어영문학전공: A(0.85), I(0.75), S(0.65) ← 거의 동일
```

**문제**: 언어/문학 전공들이 거의 동일한 프로필
**제안**: 
- 중국어/일본어: S를 조금 더 강조 (비즈니스/통역 특성)
- 국어국문학: A를 더 강조 (문학 창작/비평 특성)
- 영어영문학: E를 약간 강조 (국제 비즈니스 특성)

#### 그룹 2: 경영 관련 전공 (차별화 부족)
```
경영학전공: E(0.95), I(0.75), C(0.8)
글로벌비즈니스학전공: E(0.95), S(0.8), I(0.7)  ← E 동일
국제통상학전공: E(0.9), S(0.75), I(0.7)  ← E 거의 동일
```

**문제**: 경영 관련 전공들의 E 값이 모두 높고 유사
**제안**:
- 글로벌비즈니스학: S를 더 강조 (0.85), E는 약간 낮춤 (0.9)
- 국제통상학: I를 더 강조 (0.75), E는 약간 낮춤 (0.85)

#### 그룹 3: 공학 전공 (일부 차별화 부족)
```
화학공학전공: I(0.9), R(0.85), C(0.8)
신소재공학전공: I(0.9), R(0.85), C(0.8)  ← 동일!
```

**문제**: 화학공학과 신소재공학이 동일한 프로필
**제안**:
- 화학공학: R을 약간 낮추고 (0.8), C를 더 강조 (0.85)
- 신소재공학: I를 더 강조 (0.95), R은 유지

#### 그룹 4: 디자인 전공 (차별화 보통)
```
비주얼커뮤니케이션디자인: A(0.95), E(0.8), I(0.65)
영상애니메이션디자인: A(0.95), I(0.75), E(0.7)  ← A 동일
패션디자인: A(0.95), E(0.8), S(0.7)  ← A 동일
```

**문제**: 모든 디자인 전공의 A 값이 0.95로 동일
**제안**:
- 비주얼커뮤니케이션디자인: E를 더 강조 (0.85), A는 유지
- 영상애니메이션디자인: I를 더 강조 (0.8), A는 약간 낮춤 (0.9)
- 패션디자인: S를 더 강조 (0.75), A는 약간 낮춤 (0.9)

### 3.2 job_code_inform.csv와의 비교

실제 직무 데이터의 RIASEC 값과 비교하면:

| 직무 | 실제 RIASEC | 관련 전공 | 전공 RIASEC | 평가 |
|------|------------|-----------|-------------|------|
| 회계사 | 진취형(89)/관습형(83) | 회계세무학 | C(0.95), I(0.75) | ⚠️ E 부족 |
| 마케팅·광고·홍보관리자 | 진취형(89)/사회형(62) | 경영학, 디지털미디어 | E(0.95), A(0.85) | ✅ 적절 |
| 경영지원관리자 | 진취형(86)/사회형(63) | 경영학 | E(0.95), I(0.75) | ⚠️ S 부족 |
| 정보통신관리자 | 진취형(76)/탐구형(54) | 정보통신공학 | I(0.9), E(0.8) | ⚠️ E 과다 |

**문제점**:

3. 컴퓨터공학: 실제 직무는 현실형이 높지만, 전공은 R 부족

## 4. 50개 전공 구분 가능성 평가

### 4.1 현재 시스템의 구조

```
1단계: 계열 탐색 (10문항) → 7개 계열로 필터링
2단계: RIASEC 진단 (60문항) → 6차원 점수
3단계: 코사인 유사도 계산 → 전공 추천
```

### 4.2 차별화 가능성 분석

#### 양호한 차별화 (약 30개 전공)
- 인문대학: 사학, 미술사학, 철학, 문헌정보학, 문예창작
- 사회과학대학: 행정학, 정치외교학, 경제학, 응용통계학, 법학
- 경영대학: 경영정보학
- 공과대학: 대부분의 전공 (기계, 컴퓨터, 전기 등)
- 예술/스포츠: 디자인, 음악, 체육, 스포츠산업

#### 차별화 부족 (약 20개 전공)
- 언어/문학 전공 4개: 중어중문, 일어일문, 국어국문, 영어영문
- 경영 관련 전공 3개: 경영학, 글로벌비즈니스, 국제통상
- 공학 전공 2개: 화학공학, 신소재공학
- 디자인 전공 4개: 비주얼, 영상애니, 패션, 인더스트리얼
- 기타 유사 전공들

### 4.3 구분 가능성 평가

| 평가 항목 | 점수 | 평가 |
|----------|------|------|
| **계열 간 차별화** | ⭐⭐⭐⭐⭐ (5/5) | 7개 계열 간 차별화는 매우 양호 |
| **계열 내 차별화** | ⭐⭐⭐ (3/5) | 일부 계열 내 전공 간 차별화 부족 |
| **RIASEC 차원 활용** | ⭐⭐⭐⭐ (4/5) | 6-7차원을 효과적으로 활용 |
| **전체 구분 가능성** | ⭐⭐⭐ (3/5) | 50개 전공 중 약 60-70% 구분 가능 |

**결론**: 현재 매핑만으로는 50개 전공을 완벽하게 구분하기 어렵지만, 계열 탐색 단계와 결합하면 **70-80% 구분 가능**하다고 평가됩니다.

## 5. 개선 제안

### 5.1 즉시 개선 권장 항목

#### 1. 언어/문학 전공 차별화
```typescript
'중어중문학전공': { A: 0.8, I: 0.7, S: 0.75, E: 0.5, C: 0.5 },  // S 강조
'일어일문학전공': { A: 0.8, I: 0.7, S: 0.75, E: 0.5, C: 0.5 },  // S 강조
'국어국문학전공': { A: 0.9, I: 0.75, S: 0.6, C: 0.5, E: 0.4 },  // A 더 강조
'영어영문학전공': { A: 0.85, I: 0.75, S: 0.7, E: 0.6, C: 0.5 }, // E 약간 강조
```

#### 2. 경영 관련 전공 차별화
```typescript
'경영학전공': { E: 0.95, I: 0.75, C: 0.8, S: 0.65, A: 0.3 },     // S 추가
'글로벌비즈니스학전공': { E: 0.9, S: 0.85, I: 0.7, C: 0.75, A: 0.4 }, // S 강조, E 낮춤
'국제통상학전공': { E: 0.85, I: 0.75, S: 0.7, C: 0.65, A: 0.4 }, // I 강조, E 낮춤
```

#### 3. 공학 전공 차별화
```typescript
'화학공학전공': { I: 0.9, C: 0.85, R: 0.8, E: 0.5, A: 0.3 },     // C 강조, R 낮춤
'신소재공학전공': { I: 0.95, R: 0.85, C: 0.8, E: 0.5, A: 0.2 }, // I 더 강조
```

#### 4. 디자인 전공 차별화
```typescript
'비주얼커뮤니케이션디자인전공': { A: 0.95, E: 0.85, I: 0.65, C: 0.5, S: 0.4 }, // E 강조
'영상애니메이션디자인전공': { A: 0.9, I: 0.8, E: 0.7, C: 0.6, S: 0.5 },        // I 강조, A 낮춤
'패션디자인전공': { A: 0.9, E: 0.8, S: 0.75, I: 0.6, C: 0.5 },                // S 강조, A 낮춤
```

#### 5. 직무 데이터 반영
```typescript
'회계세무학과': { C: 0.95, E: 0.8, I: 0.75, S: 0.4, A: 0.2 },    // E 추가
'경영학전공': { E: 0.95, S: 0.7, I: 0.75, C: 0.8, A: 0.3 },     // S 추가
'컴퓨터공학전공': { I: 0.95, R: 0.75, C: 0.85, E: 0.5, A: 0.2 }, // R 추가
```

### 5.2 추가 개선 제안

#### 1. V (Values) 차원 활용도 제고
현재 V 차원이 철학, 정치외교, 행정학 등 일부 전공에만 적용되어 있음. 사회과학 전공 전반에 V 차원을 더 활용할 수 있음.

#### 2. 전공 역량 데이터 연계
`majorcapacity.csv`의 전공 역량 데이터를 RIASEC 매핑에 반영:
- 예: "논리적 사고" → I, "창의적 사고" → A, "협력" → S

#### 3. 계열 내 세부 분류
계열 탐색 단계에서 더 세분화된 분류를 도입:
- 예: "인문대학" → "언어/문학", "역사/철학", "문화콘텐츠"
