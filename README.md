통합 학생 포털 시스템 - 학생 정보 관리 및 RIASEC 기반 진로-학습 통합 분석

## 🎯 프로젝트 개요

학생들이 자신의 학적 정보를 확인하고, RIASEC 모델 기반으로 적합한 전공과 직무를 탐색할 수 있는 통합 웹 애플리케이션입니다.

### 주요 기능

#### 1️⃣ 학생 정보 시스템
- **로그인 시스템**: 학번과 비밀번호를 통한 인증 (2개 테스트 계정)
- **개인신상**: 학생의 기본 정보 및 연락처 확인
- **학점이수**: 학기별 성적, 평점 그래프, 성적 목록 조회
- **수강현황**: 학기별 수강 과목 목록 및 통계 (RIASEC 프로파일 포함)

#### 2️⃣ 전공능력진단검사
- **전공 핵심역량 측정**: 6개 능력 영역 점수화
- **레이더 & 막대 차트**: 능력 프로파일 시각화
- **강점/개선사항 분석**: 개인화된 피드백
- **학생별 맞춤 가이드**: 신입생 vs 고학년 차별화
- **통합 분석 연계**: RIASEC, 수강과목과 함께 종합 분석

#### 3️⃣ RIASEC 진로 매칭 시스템
- **적응형 문항**: 1차 결과를 바탕으로 추가 문항 제시
- **RIASEC 모델**: R, I, A, S, E, C 6차원 분석
- **3지선다**: A, B, 둘 다 관심 없음(None) 선택 가능
- **개인화 결과**: 전공 Top 3, 직무 Top 18 추천

#### 4️⃣ 진로-학습-능력 통합 분석
- **실시간 연동**: RIASEC 검사 결과와 수강 과목 데이터 실시간 연계
- **과목 RIASEC 매핑**: 모든 수강 과목에 RIASEC 프로파일 부여
- **학습 패턴 분석**: 수강 과목 기반 학습 프로파일 자동 계산 (학점 가중치 적용)
- **비교 시각화**: 레이더 차트로 진로 적성 vs 학습 경험 비교
- **맞춤형 과목 추천**: 적성에 맞는 미수강 과목 Top 5 추천
- **맞춤형 직무 추천**: 진로 적성 기반 추천 직무 Top 8
- **맞춤형 학과 추천**: 진로 적성 기반 추천 학과 Top 5
- **강점 분석**: 개인화된 강점 피드백
- **결과 저장**: localStorage를 통한 검사 결과 영구 저장
- **전공능력 연계**: RIASEC + 수강과목 + 전공능력진단 3가지 통합 분석
- **통합 인사이트**: 강점 역량, 학습 패턴, 적성을 종합한 맞춤형 조언

### 테스트 계정

로그인 페이지에서 다음 세 가지 시나리오를 체험할 수 있습니다:

- **학번**: `2301001`
- **비밀번호**: `business123`
- **특징**: 진로-학습 통합 분석 체험 가능

#### 2️⃣ 경영정보학과 2학년 (전공 선택 후)

- **특징**: 데이터/IT 기반 진로 탐색 체험

#### 3️⃣ 무전공 1학년 신입생 (수강 이력 거의 없음)
- **학번**: `2501001`
- **비밀번호**: `freshman123`
- **특징**: RIASEC 검사만으로 진로 탐색 체험

## 🎨 주요 페이지

### 1. 로그인
- 학번/비밀번호 인증
- 테스트 계정 안내
- 에러 처리

### 2. 대시보드
- 요약 통계 (평점평균, 백분위점수, 취득학점, 학년)
- 빠른 메뉴 카드
- 최근 활동 목록

### 3. 개인신상
- 기본 정보 (학번, 성명, 생년월일 등)
- 학적 정보 (학과, 학년, 학적상태)
- 연락처 정보 (전화번호, 이메일)
- 주소 정보

### 4. 학점이수
- 평점평균, 백분위점수 요약
- 학기별 평점 그래프
- 상세 성적 목록 테이블
- 통계 요약

### 5. 수강현황
- 학기별 수강 과목 목록
- 이수 구분별 통계
- 과목 상세 정보 (강좌번호, 담당교수, 시간/강의실)

### 6. 전공능력진단 📋 (NEW!)
- **전공 핵심역량**: 6개 능력 영역 측정 (경영학과/무전공)
- **종합 점수**: 전체 점수 및 백분위 표시
- **레이더 & 막대 차트**: 능력 프로파일 시각화
- **능력별 상세 결과**: 점수, 백분위, 수준(우수/보통/개선필요)
- **강점/개선사항**: 개인화된 피드백
- **학생별 가이드**: 신입생 vs 고학년 맞춤 조언

### 7. 진로-학습-능력 통합 분석 💡
- **검사 연동**: RIASEC 검사 미완료 시 검사 유도 페이지 표시
- **실시간 분석**: 검사 완료 시 자동으로 분석 페이지로 이동
- **레이더 차트**: 진로 적성(파란색) vs 학습 경험(하늘색) 비교
- **상위 차원 분석**: 진로 적성 Top 3, 학습 경험 Top 3
- **강점 분석**: 개인화된 강점 피드백
- **적성 기반 학과 추천 Top 5**: 매칭도와 추천 이유 표시
- **적성 기반 과목 추천 Top 5**: 매칭도와 이유 표시
- **적성 기반 직무 추천 Top 8**: 매칭도 기반 추천
- **전공능력 연계**: 전공능력 강점 역량과 RIASEC 적성 연계 분석
- **통합 인사이트**: 강점 역량 + 학습 패턴 + 진로 적성 종합 조언

### 8. 진로매칭 (RIASEC)
- 80개 문항 기반 적응형 검사
- RIASEC 6차원 분석
- 레이더 차트 및 결과 시각화
- 전공 및 직무 추천

## 🌟 주요 기능 플로우

### 진로-학습-능력 통합 분석 사용 시나리오

1. **로그인** → 대시보드에서 "RIASEC 검사 시작하기" 배너 확인
2. **전공능력진단** 페이지 → 전공 핵심역량 결과 확인
3. **진로매칭** 페이지 → 29개 기본 문항 + 적응형 추가 문항 응답
4. **검사 완료** → 자동으로 "진로-학습 분석" 페이지로 이동
5. **3가지 통합 분석 확인**:
   - **RIASEC 진로 적성**: 6차원 프로파일
   - **수강 과목 패턴**: 학습 프로파일 (학점 가중치)
   - **전공능력진단**: 6개 핵심역량 점수
   - 레이더 차트로 적성 vs 학습 경험 비교
   - 전공능력 강점 역량과의 연계
   - 통합 인사이트 및 맞춤형 조언
   - 맞춤형 학과 추천 Top 5
   - 맞춤형 과목 추천 Top 5
   - 맞춤형 직무 추천 Top 8
6. **과목 수강** → 추천 과목 수강 후 프로파일 변화 확인

### 데이터 흐름

```
[3가지 데이터 소스]
    ├→ [RIASEC 검사] → 진로 적성 프로파일 (R,I,A,S,E,C,V)
    ├→ [수강 과목] → 학습 프로파일 (과목 × 학점 가중치)
    └→ [전공능력진단] → 핵심역량 점수 (6개 영역)
          ↓
[통합 분석 페이지 (CareerInsight)]
    ├→ [레이더 차트] 진로 적성 vs 학습 경험
    ├→ [상위 차원 분석] 진로 적성 Top 3, 학습 경험 Top 3
    ├→ [강점 분석] 개인화된 강점 피드백
    ├→ [전공능력 연계] 강점 역량과 RIASEC 적성 비교
    ├→ [통합 인사이트] 3가지 데이터 종합 조언
    ├→ [학과 추천] 적성 × 학과 매칭 Top 5
    ├→ [과목 추천] 적성 × 미수강 과목 매칭 Top 5
    └→ [직무 추천] 적성 × 직무 매칭 Top 8
```
